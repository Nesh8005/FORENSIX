Forensix Sentinel is an advanced digital forensics tool **optimized for Kali Linux**, designed for CTF competitions and automated evidence analysis.

## Quick Start

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Install the package in editable mode:

   ```bash
   pip install -e .
   ```

3. Run the tool:

   ```bash
   forensix scan path/to/file
   ```
